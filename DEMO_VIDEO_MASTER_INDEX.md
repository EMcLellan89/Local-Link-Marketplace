## Video Specifications Summary

### Technical Requirements

| Aspect | Specification |
|--------|--------------|
| Resolution | 1920x1080 (1080p minimum) |
| Frame Rate | 30fps or 60fps |
| Aspect Ratio | 16:9 |
| File Format | MP4 (H.264 codec) |
| Bitrate | 8-10 Mbps |
| Audio | 48kHz, stereo, AAC codec |
| Thumbnail | 1280x720 JPG/PNG, under 2MB |
| Maximum Size | 128 GB (not an issue for most videos) |
| Maximum Length | 12 hours (way more than needed) |

## Common Issues & Solutions

### Issue: Low Click-Through Rate (under 3%)

**Solutions:**
1. Test new thumbnail designs
2. Rewrite title to be more specific or intriguing
3. Add emotion or curiosity to title
4. Use faces in thumbnails (if applicable)
5. Increase contrast in thumbnail
6. Make text larger and more readable

---

### Issue: Low Retention (viewers leave early)

**Solutions:**
1. Cut intro shorter (under 15 seconds)
2. State value proposition immediately
3. Add pattern interrupts (graphics, B-roll)
4. Speed up pacing
5. Remove fluff and tangents
6. Add chapter markers so viewers can skip to relevant sections
7. Use more B-roll over talking sections

---

### Issue: Low Engagement (few likes/comments)

**Solutions:**
1. Ask questions throughout video
2. Add polls and cards
3. Explicitly ask for likes/comments
4. Respond to every comment
5. Pin a question in comments
6. Create controversial or discussion-worthy content
7. Add CTAs in video and description

---

### Issue: Not Showing in Search

**Solutions:**
1. Research better keywords
2. Optimize title for search intent
3. Write comprehensive description
4. Add more relevant tags
5. Use keywords in video file name before upload
6. Add timestamps to description
7. Get initial engagement quickly (share widely at launch)

## Quick Tips for Success

1. **Hook in First 5 Seconds:** Tell viewers exactly what they'll learn
2. **Pattern Interrupts:** Change scene every 3-5 seconds
3. **Visual Interest:** Use graphics, B-roll, zooms, annotations
4. **Clear Audio:** Invest in a good microphone
5. **Consistent Branding:** Use same colors, fonts, style
6. **Optimize for Search:** Research keywords before creating
7. **Engage with Community:** Respond to every comment
8. **Analyze Data:** Learn from what works
9. **Batch Production:** Record multiple videos in one session
10. **Consistent Schedule:** Post regularly (weekly minimum)
