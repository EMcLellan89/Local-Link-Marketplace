## RECORDING SETUP & EQUIPMENT

### Recommended Equipment

**Screen Recording:**
- **Software:** OBS Studio (free), Camtasia, or ScreenFlow
- **Resolution:** 1920x1080 (1080p minimum)
- **Frame Rate:** 30fps or 60fps
- **Format:** MP4 with H.264 codec

**Audio Recording:**
- **Microphone:** USB condenser microphone (Blue Yeti, Audio-Technica AT2020)
- **Audio Interface:** Focusrite Scarlett (if using XLR mic)
- **Software:** Audacity (free) or Adobe Audition
- **Format:** WAV 48kHz, 16-bit minimum

**Video Recording (if showing face):**
- **Camera:** Webcam (Logitech C920 or better) or DSLR
- **Lighting:** Ring light or softbox lighting kit
- **Background:** Clean, professional, or use green screen

### OBS Studio Settings

**Video Settings:**
```
Base Resolution: 1920x1080
Output Resolution: 1920x1080
FPS: 30
```

**Output Settings:**
```
Encoder: x264
Rate Control: CBR
Bitrate: 8000 Kbps
Keyframe Interval: 2
Preset: Quality
Profile: High
```

**Audio Settings:**
```
Sample Rate: 48kHz
Channels: Stereo
```

## SCREEN RECORDING BEST PRACTICES

### Recording Techniques

1. **Mouse Cursor:**
   - Use larger cursor for visibility
   - Enable cursor highlighting
   - Move cursor smoothly and deliberately
   - Pause on important elements
   - Use cursor to guide viewer attention

2. **Window Management:**
   - Full-screen browser when possible
   - Zoom browser to 100% or 110%
   - Hide browser bookmarks bar
   - Close unnecessary tabs
   - Use clean, organized desktop

3. **Pacing:**
   - Move slowly through interfaces
   - Pause after clicks (let UI respond)
   - Don't rush through forms
   - Give viewers time to read important text
   - Speed up in post-production if needed

4. **Recording Segments:**
   - Record in 5-10 minute segments
   - Re-record mistakes rather than fixing in post
   - Take breaks to rest voice
   - Review footage after each segment
   - Keep separate project files for each segment

### Voiceover Tips

1. **Delivery:**
   - Speak clearly and at moderate pace
   - Use conversational tone (not reading)
   - Vary inflection to maintain interest
   - Pause naturally between thoughts
   - Emphasize key benefits
   - Smile while speaking (it comes through in voice)

2. **Script Reading:**
   - Practice script multiple times before recording
   - Mark breathing points
   - Highlight words to emphasize
   - Keep water nearby
   - Record in quiet room
   - Use pop filter to reduce plosives

3. **Recording Process:**
   - Record audio separately from screen capture (easier editing)
   - Do multiple takes of each section
   - Mark good takes while recording
   - Leave 2 seconds of silence before and after each take
   - Clap or make distinct sound for sync points

## THUMBNAIL DESIGN GUIDE

### Thumbnail Specifications
- **Size:** 1280 x 720 pixels (16:9 ratio)
- **Format:** JPG or PNG
- **File Size:** Under 2MB
- **Safe Zone:** Keep text and faces away from edges

### Design Principles

1. **Readability:**
   - Text must be readable at small sizes (320px wide)
   - Use high contrast (white text on dark background)
   - Limit text to 6 words maximum
   - Use bold, sans-serif fonts
   - Font size 60pt minimum

2. **Visual Hierarchy:**
   - One clear focal point
   - Use rule of thirds
   - Face should look at camera or toward text
   - Important elements in upper left (viewed first)

3. **Color Strategy:**
   - Use brand colors as base
   - Add high-contrast accent colors
   - Avoid similar colors to YouTube UI (red, white, black)
   - Use complementary colors for contrast

- Sub: "45 Mins • Everything"

#### Quick Start Video

- Background: Blurred platform screenshot

**Text:**

## VIDEO EDITING GUIDELINES

### Editing Software Recommendations
- **Professional:** Adobe Premiere Pro, Final Cut Pro
- **Intermediate:** DaVinci Resolve (free version excellent)
- **Beginner:** iMovie, Filmora

### Editing Workflow

1. **Import & Organize:**
   - Create folder structure (raw footage, audio, graphics, exports)
   - Name files descriptively
   - Import all assets
   - Create sequence at 1920x1080, 30fps

2. **Rough Cut:**
   - Lay out all clips in order
   - Remove obvious mistakes
   - Cut out long pauses
   - Sync audio if recorded separately
   - Don't worry about perfection yet

3. **Fine Cut:**
   - Trim for pacing
   - Remove "ums," "ahs," breathing sounds
   - Cut out form-filling (speed up or skip)
   - Add b-roll over talking sections
   - Create engaging rhythm

4. **Add Graphics:**
   - Lower thirds for sections
   - Text overlays for key points
   - Arrows and annotations
   - Chapter title cards
   - Zoom effects on important UI

5. **Color Correction:**
   - Consistent color across all clips
   - Slight saturation boost
   - Brightness/contrast adjustment
   - Match to brand colors

6. **Audio Mixing:**
   - Normalize voice audio
   - Add background music (quiet, 15-20% volume)
   - Sound effects for transitions
   - Fade in/out music at beginning and end
   - Duck music during important talking points

7. **Transitions:**
   - Simple cuts (most common)
   - Cross dissolve for scene changes
   - Fast cuts for energy
   - Avoid cheesy transitions
   - Match transitions to music

### Pacing Guidelines

**Full Demo (30-45 min):**
- Moderate pace
- Allow time for viewers to absorb information
- Speed up form filling to 150-200%
- Include 3-5 second pauses between major sections

**Quick Start (5-10 min):**
- Fast pace
- Frequent cuts (every 3-5 seconds)
- Speed up everything to keep moving
- Use energetic music

**Focused Videos (10-15 min):**
- Balanced pace
- Clear explanations with examples
- Natural speaking rhythm
- Speed up repetitive tasks

### B-Roll Usage
- Insert b-roll over voiceover
- Show different angles of same action
- Cut to results/outcomes
- Demonstrate real-world usage
- Break up long talking segments

## ANALYTICS & ITERATION

### Key Metrics to Track

**Engagement Metrics:**
- Average view duration (aim for 50%+)
- Click-through rate (aim for 4%+)
- Engagement rate (likes, comments, shares)
- Cards clicks
- End screen clicks
- Subscriber conversion rate

**Audience Retention:**
- Introduction retention (first 30 seconds)
- Mid-video retention
- Audience retention graph shape
- Drop-off points

**Discovery Metrics:**
- Impressions
- CTR from impressions
- Traffic sources
- Search terms used
- Suggested video performance

### Analysis Questions

**If low CTR (under 3%):**
- Is thumbnail compelling enough?
- Does title match search intent?
- Is title too vague or too complex?
- Try A/B testing different thumbnails

**If low retention (under 40%):**
- Does intro hook viewers quickly?
- Is pacing too slow?
- Are sections too long?
- Is content valuable throughout?
- Cut intro shorter and get to value faster

**If low engagement:**
- Did you ask for likes/comments/subscribes?
- Is there a clear call-to-action?
- Are you asking questions in video?
- Is content generating discussion?
- Add polls and questions

### Optimization Strategies

**After 48 Hours:**
- Review analytics
- Respond to all comments
- Add any missing information to description
- Create follow-up content based on questions

**After 1 Week:**
- Compare to similar videos
- Identify what worked well
- Note improvements for next video
- Update thumbnail if CTR is low (<3%)

**After 1 Month:**
- Deep dive into retention graph
- Identify best-performing sections
- Create standalone videos on popular sections
- Optimize for search terms that are working

**Ongoing:**
- Update descriptions with new links
- Add videos to playlists
- Create content series
- Cross-promote between videos
- Refresh thumbnails for older videos
