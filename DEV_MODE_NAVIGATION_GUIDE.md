## 🔧 HOW TO USE DEV MODE

### 1. Start the Development Server
```bash
npm run dev
```

### 2. Open Your Browser
Navigate to: `http://localhost:5173`

### 3. Use the Role Switcher
Look for the floating role switcher in the bottom-left corner:
- Click to switch between: Customer, Merchant, Partner, Admin
- Current role is highlighted
- Page will reload with new role context

### 4. Navigate Freely
- All authentication is bypassed
- All payment flows are simulated
- All database queries use mock data
- No Stripe/payment processing in dev mode

### 5. Test Features
- Create deals as a Merchant
- Browse as a Customer
- Manage partners as Admin
- Test full user journeys end-to-end

## 🚨 IMPORTANT NOTES

### Before Going Live
1. Set `VITE_DEV_MODE=false` in `.env`
2. Set `VITE_BYPASS_MODE=false` in `.env`
3. Test authentication flows
4. Test payment processing with Stripe test mode
5. Verify all RLS policies are working

### Dev Mode Features
- No authentication required
- Mock data for all entities
- Simulated payment flows
- Role switching without re-login
- Full platform access
