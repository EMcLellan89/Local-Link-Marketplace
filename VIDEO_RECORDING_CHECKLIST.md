## PRE-RECORDING SETUP

### Demo Data Preparation
- [ ] Create 3-5 sample merchant accounts
  - [ ] Restaurant (e.g., "Tony's Italian Bistro")
  - [ ] Retail (e.g., "Fashion Forward Boutique")
  - [ ] Services (e.g., "Elite Fitness Studio")
  - [ ] Entertainment (e.g., "Cinema Palace")
  - [ ] Health & Wellness (e.g., "Zen Spa & Wellness")

- [ ] Create 15-20 sample deals across categories
  - [ ] Various price points ($10-$200)
  - [ ] Different discount types (%, BOGO, Fixed Amount)
  - [ ] Mix of active, draft, and expired deals

- [ ] Create 2-3 customer accounts
  - [ ] Make 5-10 sample purchases
  - [ ] Add deals to favorites
  - [ ] Complete profile information

- [ ] Set up CRM demo data
  - [ ] 20-30 sample leads
  - [ ] Various stages in pipeline
  - [ ] Sample activities and notes

- [ ] Create sample invoices
  - [ ] 5 paid invoices
  - [ ] 2-3 pending/sent invoices
  - [ ] 1 overdue invoice

- [ ] Prepare marketing samples
  - [ ] Sample email campaign
  - [ ] Customer segment examples
  - [ ] Sample reviews and ratings

### Technical Setup
- [ ] Clear browser cache
- [ ] Set screen resolution to 1920x1080
- [ ] Close unnecessary browser tabs
- [ ] Turn off notifications
- [ ] Test microphone audio
- [ ] Prepare teleprompter/notes
- [ ] Have demo login credentials ready
- [ ] Test payment flow in test mode

### Recording Environment
- [ ] Clean desktop background
- [ ] Good lighting
- [ ] Quiet environment
- [ ] Multiple takes planned
- [ ] Screen recording software ready
- [ ] Mouse cursor highlighter enabled

## SPEAKING POINTS & SCRIPT NOTES

### Key Phrases to Use:
- "All-in-one business solution"
- "Everything you need in one platform"
- "Affordable pricing starting at just $49/month"
- "Integrated GoPayBright payment processing"
- "Complete CRM with accounting built-in"
- "Save thousands compared to using multiple tools"
- "Designed specifically for local businesses"
- "Professional features without the enterprise price tag"

### Benefits to Emphasize:
- Cost savings vs. buying separate tools
- Time savings through automation
- Ease of use
- Comprehensive feature set
- Excellent support
- Scalability (Starter to Enterprise)
- Secure and reliable
- Mobile-friendly

### Common Objections to Address:
- "Too complex" → Show how intuitive the interface is
- "Too expensive" → Compare to cost of separate tools
- "Don't need all features" → Explain tiered pricing
- "Already have tools" → Show CRM migration feature
- "Not tech-savvy" → Highlight ease of use and support

## EQUIPMENT & SETTINGS RECOMMENDATIONS

### Screen Recording:
- Resolution: 1920x1080 (1080p)
- Frame rate: 30 fps minimum
- Bitrate: High quality
- Cursor highlighting: ON
- Click animations: ON

### Audio:
- Use external microphone
- Remove background noise
- Add slight compression
- Normalize audio levels
- Remove mouth clicks/breathing

### Video:
- Use intro template
- Consistent branding
- Professional transitions
- On-screen annotations for key points
- Zoom in on important UI elements

## BACKUP PLAN

### If Demo Breaks:
- [ ] Have screenshots ready
- [ ] Pre-recorded segments as backup
- [ ] Multiple test accounts ready
- [ ] Know how to switch accounts quickly
- [ ] Have support team on standby

### If Features Don't Work:
- [ ] Acknowledge and explain upcoming feature
- [ ] Show mockup or wireframe
- [ ] Redirect to working features
- [ ] Keep recording, edit later
