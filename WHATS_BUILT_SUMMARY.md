## 🛠️ Technical Architecture

### Frontend
- **Framework**: React 18 + TypeScript
- **Routing**: React Router v7
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build**: Vite
- **Deployment**: Vercel-ready

### Backend
- **Database**: Supabase (PostgreSQL)
- **Edge Functions**: Supabase Functions (Deno)
- **AI**: OpenAI GPT-4o-mini
- **Payments**: Stripe
- **Communications**: Twilio (SMS) + SendGrid (Email)
- **Voice AI**: VAPI integration

### Database
- **Tables**: 400+
- **RLS**: Enabled on all tables
- **Indexes**: Optimized for performance
- **Migrations**: 200+ migration files
- **Functions**: 50+ database functions

### AI Workforce
- **Events**: Automatic job creation
- **Jobs**: Queue with retry logic
- **Bots**: 18 specialized bots
- **Circuit Breaker**: Auto-throttling
- **DLQ**: Failed message recovery
- **Health Monitoring**: Real-time metrics

### Security
- Row Level Security (RLS) on all tables
- Admin-only policies
- Partner territory restrictions
- Merchant data isolation
- Audit logging
- HTTPS everywhere
- JWT authentication
- API key management

### Performance
- Foreign key indexes
- Query optimization
- Caching strategies
- CDN delivery
- Code splitting
- Lazy loading
- Image optimization
